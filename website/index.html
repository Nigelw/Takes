Frequently asked questions

Does Takes work with streaming services?

Takes supports Apple Music, Spotify, and YouTube in a slightly creative way. To work around streaming limitations, when you give it Apple Music and Spotify URLs it automatically sources a version from YouTube.

How much does Takes cost?

Takes is free and open source. You can download it, use it, and inspect the code on GitHub.

Why did you build Takes?

I built Takes to scratch my own itch. I've always wanted a tool like Kaleidoscope, but for audio files. I love hearing the differences between original releases and remasters, and I wanted an app that lets me listen and toggle between them. I built a quick utility, then decided to turn it into a polished app because I love old school native apps that sweat the details. I'm sharing it in case others find it useful too.

Do you offer support?

Just the help section below.

Help

Getting audio into Takes

  • Drag files and folders onto the app, use Quick Open to grab your current selection from Finder or Apple Music, or use the Open dialog box like a normie.
  • You can also open streaming music. Give it a URL from Spotify, Apple Music, or YouTube and you're good to go.
  • Takes supports up to 32 loaded audio files at once.
  • Takes plays a wide range of audio formats.

Selection looping

Use selection looping to focus on a specific passage. Drag or Shift+Click in the timeline area to create a loop. "Switch & Repeat" mode is automatically enabled when looping. This changes tracks eachtime playback loops so you can compare the same passage across tracks.

  • You can also repeat one, to keep looping the same track.
  • Clearing a loop automatically switches repeat back off.

Blind listening mode

Decide which take sounds best without the bias of knowing which version you're listening to. Blind listening mode hides names and metadata, anonymizes the waveforms, and shuffles the track order. It even hides the track name from the Now Playing widget in Control Center. No cheating! 😜

  • Turn off blind listening mode to reveal your the name of your favorite. The track order stays in place when turning it off.

Auto-align

Different copies of a track don't always line up perfectly. An extra bit of silence at the start of one can make toggling back and forth jarring, as each switch between tracks jumps the music backward and forward.

Auto-align analyzes the audio to find matching sections, and adjusts offsets so tracks line up perfectly.

  • If tracks fall out of sync over time due to a difference in recordings, you can re-run auto-align at any time. It sync offsets based on the current playhead position.
  • Auto-align only accepts confident matches and avoids false positives, so it won't set crazy offsets if file matching is ambiguous.
  • For tracks with larger tempo differences, auto-align offers a deeper analysis mode that searches tempo deviations, up to about 6 percent, before applying an alignment.
  • If auto-align can't sync up your tracks, Takes is designed to make manual alignment as fast as possible. A great workflow is to start playback, nudge alignment, toggle back and forth between tracks, and repeat if necessary.
  • The Takes UI is specifically designed for quick keyboard navigation to make this easy: tab into the offset field, nudge the offset with Shift + ↑/↓, hit tab or enter, hit X to toggle tracks.
