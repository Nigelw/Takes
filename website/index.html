Frequently asked questions

Does Takes work with streaming services?

No. Takes is for local file playback only. Bring your own audio files and compare them side by side in the app.

How much does Takes cost?

Takes is free and open source. You can download it, use it, and inspect the code on GitHub.

Why did you build Takes?

I built Takes to scratch my own itch. I love hearing the differences between original releases and remasters, and I've always wanted an app that lets me listen and toggle between them. I built a small utility, then decided to turn it into a polished app because I love old school native apps that sweat the details. I'm sharing it in case others find it useful too.

Do you offer support?

Just the help section below.

Help

Getting audio into Takes

  • Open one or more local files with Cmd O, or drag files anywhere into the window.
  • Use Shift Cmd F for Quick Open from Finder.
  • Use the + menu above the track list for Quick Open from Apple Music.
  • Drop a file on a specific row to replace that track, or drag rows to reorder them.
  • Takes supports up to 32 loaded local audio files at once.

Selection looping

Drag or Shift Click in the timeline area to constrain playback to the selected range. Use it to focus on a specific passage and loop across tracks.

  • Drag selection handles to resize the loop.
  • Repeat Off stops at the end of the current playable range.
  • Repeat One restarts the current track from the start of that range.
  • Switch & Repeat advances to the next track on each wrap so you can compare the same passage across tracks.
  • Creating a loop automatically switches Takes into Switch & Repeat. Deselecting the loop turns repeat back off.

Blind listening mode

Blind Listening Mode is for comparing takes without knowing which source file is which. It hides names and metadata, anonymizes the waveforms, and shuffles the track order.

  • Use it when you want your ears to decide without being influenced by knowing which version you're listening to. It even hides the filename from the Now Playig widget in Control Center. No cheating!
  • Turn it on, switch between tracks, pick your favorite, then reveal the names afterward.

Auto-Align

Auto-Align listens to the audio itself and computes per-track offsets so similar sections line up. The quick pass compares audio-change patterns and only accepts confident matches, which helps it avoid false positives when files are unrelated or too ambiguous.

  • Auto-Align matches based on the playhead position. Re-run it at any time if tracks drift over time and you need to realign them.
  • If the quick pass cannot find a clean match, Takes leaves that track unaligned instead of forcing it.
  • For slightly different speeds or tempo drift, it can offer a deeper analysis that searches tempo deviations, up to about 6 percent, before applying an alignment.
