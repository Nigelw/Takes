Takes is a macOS app for comparing audio tracks with instant A/B switching, waveform alignment, and blind listening mode.

## Build From Source

Requirements:

- macOS
- Xcode with command line tools selected

Open the project:

```bash
open Takes.xcodeproj
```

In Xcode, select the `Takes` scheme, choose `My Mac` as the run destination,
and press `Cmd-R`.

Developer and agent workflow notes live in [AGENTS.md](AGENTS.md). User help
and release notes live on the [Takes website](https://takes.nigelwarren.com).
